## Usage
After installation, to run the project, follow these steps:
1. Navigate to the project directory:
   ```sh
   cd virtualMemory
   ```
2. Run the script:
   ```sh
   python main.py
   ```
3. A resulting file, `output-dp.txt`, will be generated in the current directory.
